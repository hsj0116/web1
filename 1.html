HTML

Hypertext Markup Language (HTML) is the standard markup languge for creating web pages and web applications. In HTML, the
element creates a line break. You can add it wherever you want text to end on the current line and resume on the next.

The HTML line break element can be used to display poems, song lyrics, or other forms of content in which the division of lines is significant.
